# 마이크로바이옴에 대해 알아야 할 모든 것
### Everything You Need to Know About the Microbiome

혐기성 세균(anaerobic bacteria)을 **그람염색 + 형태(구균/간균)** 기준으로 **GPAC · GPAB · GNAC · GNAB** 4대 분류로 나누고, 각 군의 대표 세균 · 관련 질병 · 오염 가능 식품 · 항생제 감수성, 그리고 구취(VSC) 기전까지 정리한 임상 참조 문서입니다.

A clinical reference that classifies anaerobic bacteria into the four major groups **GPAC / GPAB / GNAC / GNAB** by Gram stain and morphology, and organizes each group's representative species, associated diseases, potentially contaminated foods, antibiotic susceptibility, and the mechanism of halitosis (VSC).
