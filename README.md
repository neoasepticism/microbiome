# Everything You Need to Know About the Microbiome

A clinical reference that classifies anaerobic bacteria into the four major groups **GPAC / GPAB / GNAC / GNAB** by Gram stain and morphology (cocci/bacilli), and organizes each group's representative species, associated diseases, potentially contaminated foods, antibiotic susceptibility, and the mechanism of halitosis (volatile sulfur compounds, VSC).

## Contents

- Definitions and phylogeny of the four anaerobe classes (GPAC/GPAB/GNAC/GNAB)
- Five representative species, major associated diseases, and potentially contaminated foods for each class
- Antibiotic susceptibility tables and a cross-class summary
- Deep dives on key clinical pathogens such as *Finegoldia magna*
- Volatile sulfur compounds (VSC) behind halitosis and the foods that serve as their substrate
